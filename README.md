# Менеджер задач (Task Manager)

![Python](https://img.shields.io/badge/Python-3.11-blue.svg)
![Django](https://img.shields.io/badge/Django-5.1-green.svg)

Веб-приложение для управления задачами, разработанное на Django. Позволяет создавать и отслеживать задачи, управлять их статусом и приоритетами.

## 🚀 Возможности

- ✅ Просмотр списка всех задач
- 📝 Создание новых задач
- 🔄 Отслеживание статуса выполнения
- 🔍 Фильтрация активных задач
- 💫 Простой и удобный интерфейс

## 🛠 Технологии

- **Python 3.11+**
- **Django 5.1** - современный веб-фреймворк
- **SQLite** - база данных
- **Bootstrap** - отзывчивый дизайн
- **Ruff** - инструмент для проверки кода

## ⚙️ Установка и запуск

### Предварительные требования

- Python 3.11 или выше
- UV (современный Python package installer)

### Шаги установки

1. Клонируйте репозиторий и перейдите в директорию проекта:
```bash
git clone [url-репозитория]
cd taskmanager
```

2. Создайте виртуальное окружение и активируйте его:
```bash
python -m venv venv
.\venv\Scripts\activate
```

3. Установите зависимости с помощью uv:
```bash
uv pip sync requirements.txt
```

4. Настройте базу данных:
```bash
python manage.py migrate
```

5. Запустите сервер разработки:
```bash
python manage.py runserver
```

Приложение будет доступно по адресу: http://127.0.0.1:8000/

## 🔧 Разработка

### Проверка кода

В проекте используется Ruff для обеспечения качества кода. 

```bash
# Проверка стиля и линтинг
ruff check .

# Автоматическое форматирование
ruff format .
```
